# 测试工具

用于测试数据库和系统功能的工具。

## 工具列表

### test_database.py - 数据库测试

全面测试数据库的连接、结构、数据完整性等。

**使用方式**：
```bash
python tools/testing/test_database.py
```

**测试项目**：
- ✅ 数据库文件存在性
- ✅ 表结构正确性
- ✅ 数据完整性
- ✅ JSON格式正确性
- ✅ 查询性能

## 运行测试

```bash
cd /root/projects/object_attributes_annotation_tool/modular_version
```
